Detailed Description

template<VkFormat FORMAT>
class foray::util::ImageLoader< FORMAT >

General purpose image loader.

Supported File Types

  • PNG, JPG, BMP, HDR via StbImage (plus a few other, see <tinygltf/stb_image.h>)
  • EXR via TinyExr
