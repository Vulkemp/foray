Detailed Description

Provides sampler objects based on VkSamplerCreateInfo specifications.

Remarks
Exclusively designed to work with SamplerReference to ensure proper reference counting! Works only for VkSamplerCreateInfo without .pNext set Will count references to a unique sampler type, and automatically reuse sampler types and destroy
